The files FinanceScraper.py and Sports_Scraper.py were used to gather the data. They have an extreme runtime (hours, at least).The file Group4Cleaner.py was used to format and clean the raw data scraped from the web. It has a more manageable runtime (10 minutes or so).The file Group_5_Project_2.py performs all the basic analyses and the first three hypotheses tests.The file “Group 5 Project 3.py” performs the final hypothesis test and creates all the plotly visualizations.
